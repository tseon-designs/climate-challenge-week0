# Notebooks

This folder contains all the Jupyter notebooks for the climate analysis project.

## Structure

| Notebook | Description |
|----------|-------------|
| `eda_ethiopia.ipynb` | Data profiling, cleaning, and EDA for Ethiopia |
| `eda_kenya.ipynb` | Data profiling, cleaning, and EDA for Kenya |
| `eda_sudan.ipynb` | Data profiling, cleaning, and EDA for Sudan |
| `eda_tanzania.ipynb` | Data profiling, cleaning, and EDA for Tanzania |
| `eda_nigeria.ipynb` | Data profiling, cleaning, and EDA for Nigeria |
| `compare_countries.ipynb` | Cross-country comparison and vulnerability ranking |

## How to Run

```bash
# Activate virtual environment first
.venv\Scripts\activate   # Windows
source .venv/bin/activate  # macOS/Linux

jupyter notebook
```

Then open any notebook from the browser interface.
